\# 企業資源管理系統 (CompanyAPP)

本專案為基於 ASP.NET Core 9.0 架構開發的企業資源管理平台 (ERP)，旨在提供廠商資料維護、人力資源管理以及任務派工的整合解決方案。系統採用 MVC 設計模式，並實作完整的權限控管 (RBAC) 與資料視覺化功能。


\## 技術架構


本系統採用以下技術堆疊進行開發：


\*   \*\*開發框架\*\*: .NET 9.0 (ASP.NET Core MVC)

\*   \*\*資料庫\*\*: SQLite (經由 Entity Framework Core 實作 Code First)

\*   \*\*前端介面\*\*: Bootstrap 5, JavaScript (ES6), jQuery, HTML5/CSS3

\*   \*\*身分驗證\*\*: ASP.NET Core Identity

\*   \*\*數據視覺化\*\*: Chart.js

\*   \*\*開發環境\*\*: Visual Studio 2022 Community


\## 功能模組說明

\### 1. 系統帳號與權限管理

實作基於角色的存取控制 (Role-Based Access Control, RBAC)。

\*   \*\*身分驗證\*\*：支援本地帳號註冊與登入，並已將錯誤訊息與介面完整中文化。

\*   \*\*角色分級\*\*：系統區分為「管理員 (Admin)」與「一般用戶 (User)」。

\*   \*\*後台管理\*\*：僅授權管理者進入後台進行角色分配。

\*   \*\*安全機制\*\*：針對系統預設的最高管理者 (Super Admin) 實作防刪除與降級保護機制。

\### 2. 廠商管理模組 (Vendors)

提供供應商資料的生命週期管理。

\*   \*\*基本資料維護\*\*：支援標準 CRUD (新增、讀取、更新、刪除) 操作。

\*   \*\*多媒體整合\*\*：實作圖片上傳功能，並於編輯介面提供即時預覽 (Instant Preview)。

\*   \*\*關聯檢視\*\*：於廠商詳情頁面可直接瀏覽該廠商旗下的所有員工清單。

\### 3. 員工管理模組 (Employees)

管理企業內部或外包人員資訊。

\*   \*\*資料關聯\*\*：實作資料庫外鍵關聯 (Foreign Key)，新增員工時需透過動態下拉選單指定所屬廠商。

\*   \*\*非同步搜尋\*\*：利用 AJAX 技術實作即時搜尋功能，支援多欄位關鍵字過濾（姓名、職位、Email）。

\*   \*\*批次處理\*\*：支援列表多選功能，可執行批次刪除操作。

\*   \*\*派工整合\*\*：於員工詳情頁提供快捷派工入口，系統將自動帶入員工參數。

\### 4. 任務派工模組 (Missions)

提供工作任務的指派與狀態追蹤。

\*   \*\*狀態管理\*\*：支援任務生命週期狀態切換（待處理、處理中、已完成）。

\*   \*\*智慧表單\*\*：實作連動式輸入介面，使用者可從預設清單選擇或切換至自定義輸入模式。

\*   \*\*時效警示\*\*：系統自動計算任務截止日期，針對即將到期的任務進行視覺化警示。

\### 5. 數據儀表板 (Dashboard)

\*   整合首頁儀表板，利用 Chart.js 繪製企業人力分布統計圖表，提供管理者即時決策數據。


\## 安裝與執行指南

若需於本地環境部署此專案，請參照以下步驟：

\### 1. 取得專案代碼

git clone \[專案 Repository URL]
cd CompanyAPP


\### 2. 資料庫遷移 (Migrations)

本專案使用 SQLite 資料庫。初次執行前，需確保本地資料庫結構已建立。請於 Visual Studio 的「套件管理器主控台 (Package Manager Console)」執行以下指令：

Update-Database


\### 3. 啟動應用程式

按下 F5 或執行以下指令啟動伺服器：

dotnet run

應用程式預設將運行於 http://localhost:5xxx 。


\## 系統初始化設定

系統首次啟動時，將透過 Seed Data 自動建立第一位使用者最高權限管理員帳號：

\*   \*\*帳號 (Email)\*\*: XxxxxxX@xxx.tw

\*   \*\*權限\*\*: Administrator (Admin)
